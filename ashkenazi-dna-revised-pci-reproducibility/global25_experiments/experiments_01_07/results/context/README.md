# Experiment 1 Context Tables

`01_reddit_screenshot_14_targets.csv` is a literal transcription of the fourteen Calabrian and Sicilian rows and the displayed Average row in the Reddit screenshot preserved as `figures/article_context/reddit_southern_italian_screenshot.png`. Its short display aliases and seven-source panel are not treated as equivalent to the declared sources used in the controlled replication.

`01_pooled_ashkenazi_matched_result.csv` records the separate pooled Ashkenazi run made with the same seven declared ancient sources as expanded Experiment 1. The pooled target is not one of the 196 rows in `results/raw/01_controlled_prehistoric_expanded_results.tsv`; it is kept here so the article can compare the pooled result without misdescribing the row count of the expanded run.

The derivation script uses both context files only for the explicitly labeled comparisons. It does not merge them into the raw 196-target calculator export.
