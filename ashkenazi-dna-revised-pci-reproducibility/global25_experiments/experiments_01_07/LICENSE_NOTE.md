# Licensing and Reuse Note

No blanket license is asserted over third-party Global25 coordinate data in this repository. The coordinate rows remain subject to the terms, attribution requirements and permissions of their original compilers and upstream sources.

The repository’s original explanatory documentation and validation script are provided for transparent scholarly reproduction. Before redistributing or incorporating the coordinate data into another project, verify the licensing and citation requirements of the original data providers.

This file is intentionally not an open-source license for the underlying coordinate dataset.
