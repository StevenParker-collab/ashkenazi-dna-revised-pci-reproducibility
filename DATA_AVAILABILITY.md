# Data and software availability

No newly sequenced genotype data are reported. The repository distributes author-generated specifications, derived tables, raw copied outputs, screenshots, source and target coordinate lines where retained, and validation code.

## qpAdm and FST

The reported qpAdm and FST analyses were run through the IllustrativeDNA Admix Lab web interface using Human Origins v62.0. The repository identifies the exact target, source, right-set, comparator labels, and settings and preserves copied outputs and screenshots.

The underlying Human Origins genotype files are not redistributed. Reproduction therefore requires access to the same web environment and dataset version. A command-line ADMIXTOOLS analysis using a separately obtained compatible dataset would be an independent replication, not the original workflow.

## Global25 Experiments 1-15

The experiment suite uses scaled Global25 population-average coordinates and the ExploreYourDNA Atlantic Bronze Age calculator. The documented settings are cycles 0.25X, distance column off, printed zeroes off, fast mode off, and aggregate mode on, except where a supplied setup records only the settings visible in its screenshot.

Experiments 1-7 contain paste-ready panels and raw or derived results. Experiments 10-15 contain supplied or cross-linked source and target panels, setup notes, raw outputs, comparisons, and screenshots. For Experiments 8 and 9, the supplied handoff preserved raw output tables and screenshots but did not include separate coordinate files for every ancient target. The available evidence is retained as supplied, and no missing coordinate row has been reconstructed from an unverified external source.

The Experiment 14 analysis directory contains the full-dimensional Ashkenazi Germany vector calculation, exact block inputs, result table, direct-distance ranking, verification script, and illustrative PCA figure. The script reads the distributed Experiment 14 source, target, and raw-result files directly.

## Interpretation

Global25 mixture outputs are descriptive constrained similarity weights conditional on the declared source panel. They are not formal qpAdm estimates, literal ancestry percentages, confidence intervals, or admixture dates.
