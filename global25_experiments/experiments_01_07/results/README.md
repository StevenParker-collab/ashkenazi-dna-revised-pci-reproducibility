# Results Directory

- `raw/` contains canonical calculator exports. Expanded Experiment 1 is a 196-target TSV; Experiments 2–7 retain their existing CSV outputs.
- `context/` contains the literal Reddit screenshot transcription and the separate pooled Ashkenazi matched result. Neither is silently inserted into the 196-row raw export.
- `derived/` contains tables regenerated by repository scripts.
- `summary.csv` is the cross-experiment high-level index.

The calculator's final `Average` row is preserved when it appears, but it is not treated as a population-weighted estimate. This matters especially in expanded Experiment 1 because the cohort sizes are unequal.

Regenerate and check the Experiment 1 summaries with:

```bash
python3 scripts/derive_natufian_math.py
python3 scripts/derive_natufian_math.py --check
```

When adding or replacing a result, preserve all target labels and displayed decimals, update the matching documentation and manifest, regenerate derived files and hashes, and run `python3 scripts/validate_repository.py`.
